Multiple columns

The CSS multi-column layout allows easy definition of multiple columns of text.
To use Multi-column you use the properties:

  • column-count
  • column-gap
  • column-rule-style
  • column-rule-width
  • column-rule-color
  • column-rule
  • column-span
  • column-width

    • column-count

    The column-count property specifies the number of columns an element should be divided into.
