Shadows properties

You can add shadow to text and to elements with the properties:

  • text-shadow
  • box-shadow

    • Text-shadow

    Text-shadow is a propertie to apply on the text.
    This propertie take 4 argument which is:

    text-shadow: horizontal shadow, vertical shadow, blur effect, color
    For exmple:

    text-shadow: 2px 2px 5px cyan;

    If you want more exemple, you can see some other on this link: Text-shadow exmple

    Box-Shadow

    The CSS box-shadow property is used to apply one or more shadows to an element.
    Box-shadow take the same parameters than text-shadow with a inset parameter.
    The inset parameter changes the shadow from an outer shadow (outset) to an inner shadow.

    box-shadow: horizontal shadow, vertical shadow, blur effect, color, inset;
    Exemple without inset:

    box-shadow: 10px 10px 5px 12px lightblue

    Exemple with inset:

    box-shadow: 10px 10px 5px 12px lightblue inset
