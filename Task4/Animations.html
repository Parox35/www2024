Animations

With CSS you can do some animation without javascript with this properties:

  • @keyframes
  • animation-name
  • animation-duration
  • animation-delay
  • animation-iteration-count
  • animation-direction
  • animation-timing-function
  • animation-fill-mode
  • animation

    • @keyframes

    When you specify CSS styles inside the @keyframes rule, the animation will gradually change from the current style to the new style at certain times. To get an animation to work, you must bind the animation to an element.

    Change the background of a div in four time

    To apply an animation you need to use the propertie animation-duration to see a modification and the propertie animation-name with the name of the animation in the keyframes you give.

    @keyframes backgroundAnimation{
    0% {background-image: linear-gradient(to bottom right, white, red);}
    25% {background-image: linear-gradient(to bottom right, red, blue);}
    50%{background-image: linear-gradient(to bottom right, blue, yellow);}
    100% {background-image: linear-gradient(to bottom right, yellow, green);}
    }

    #animationDIV{ width: 100px;
    height: 100px;
    margin: 1%;
    animation-name: backgroundAnimation infinite;
    animation-duration: 4s;
    animation-iteration-count: infinite;
    }
