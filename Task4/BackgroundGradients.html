Background gradients properties

It exist three type of background gradients which are:

  • Linear Gradients (goes down/up/left/right/diagonally)
  • Radial Gradients (defined by their center)
  • Conic Gradients (rotated around a center point)

    • Linear gradients

    To create a linear gradient you must define at least two color stops. Color stops are the colors you want to render smooth transitions among.
    You can also set a starting point and a direction (or an angle) along with the gradient effect.
    To use this propertie you need to use this synthaxe:

    background-image: linear-gradient(direction, color-stop1, color-stop2, ...);
    For example, if you want to do a gradient of cyan to purple going right to left:

    background-image: linear-gradient(to right, cyan, purple)

    You can also do diagonally:

    background-image: linear-gradient(to bottom left, Red, White)

    If you want more example, you can click on this link: CSS Linear Gradients

    Radial gradient

    A radial gradient is defined by its center with at least two color step.
    To use this propertie you need to use this synthaxe:

    background-image: radial-gradient(shape size at position, start-color, ..., last-color);
    For example:

    background-image: radial-gradient(circle, red, yellow, green);

    It exists four size parameters to define the size of the gradient. It can take four values (closest-side, farthest-side, closest-corner, farthest-corner)
    For more example, you can click on this link: CSS Radial Gradients

    Conic Gradients

    A conic gradient is a gradient with color transitions rotated around a center point.
    To use it you need at least two different color.
    To use this propertie you need to use this synthaxe:

    background-image: conic-gradient([from angle] [at position,] color [degree], color [degree], ...);

    background-image: conic-gradient(red 0deg, red 90deg, yellow 90deg, yellow 180deg, green 180deg, green 270deg, blue 270deg);
