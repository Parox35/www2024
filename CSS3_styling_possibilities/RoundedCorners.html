Rounded corners

Border-radius Propertie:

A shorthand property for setting all the four border-*-*-radius properties

You can set all the border with the same size like bellow:

border-radius: 25px;
border: 2px solid black;

You also set different size for each border: (first value applies to top-left corner, second value applies to top-right corner, third value applies to bottom-right corner, and fourth value applies to bottom-left corner)

border-radius: 25px 30px 50px 60px;
border: 2px solid black;

Two values use (first value applies to top-left and bottom-right corners, and the second value applies to top-right and bottom-left corners):

border-radius: 20px 50px;
border: 2px solid black;

If you want to applies a border on just one corner use the properties bellow:

border-top-left-radius properties:

Defines the shape of the border of the top-left corner

border-top-left-radius: 25px;
border: 2px solid black;

border-top-right-radius propertie:

Defines the shape of the border of the top-right corner

border-top-right-radius: 25px;
border: 2px solid black;

border-bottom-right-radius propertie:

Defines the shape of the border of the bottom-right corner

border-bottom-right-radius: 25px;
border: 2px solid black;

border-bottom-left-radius propertie:

Defines the shape of the border of the bottom-left corner

border-bottom-left-radius: 25px;
border: 2px solid black;
