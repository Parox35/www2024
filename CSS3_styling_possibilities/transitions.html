Transitions properties

CSS transitions allows you to change property values smoothly, over a given duration.
There is multiple properties like:

  • transition
  • transition-delay
  • transition-duration
  • transition-property
  • transition-timing-function

    • We will just show the transition propertie.

    If you want some example from the other properties: CSS Transitions

    Transition propertie

    To use this propertie you need to specify to things:

  • The effect you want
  • The duration of this effect

    • transition: property to effect, delay;
    For example grow a div from 100px*100px to 200px*200px:
    transition: width 2s, height 2s;
